# Customer Expense Analysis using Power BI

## Project Description
This project analyzes customer expense data to understand spending patterns
using Python and Power BI.

## Tools & Technologies
- Python
- Pandas
- Power BI
- CSV Dataset

## Project Structure
- data/expenses.csv : Raw expense data
- analysis/expense_analysis.py : Python analysis script
- Customer_Expense_Dashboard.pbix : Power BI Dashboard

## Key Insights
- Total customer spending
- Average spending per customer
- Category-wise spending
- Monthly expense trend
- Payment method analysis

## Outcome
An interactive Power BI dashboard that helps visualize customer spending behavior.
